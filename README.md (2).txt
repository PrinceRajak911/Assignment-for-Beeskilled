# ??? Temperature Converter

A simple Python command-line utility to convert temperatures between Celsius and Fahrenheit. This project was built to demonstrate the use of basic mathematical formulas, user input handling, and modular functions in Python.

## ?? Features

* **Celsius to Fahrenheit Conversion:** Uses the formula `(C * 9/5) + 32`.
* **Fahrenheit to Celsius Conversion:** Uses the formula `(F - 32) * 5/9`.
* **Error Handling:** Gracefully handles invalid text inputs when numerical values are expected.
