# ?? Student Grade Calculator

A Python command-line application that takes student marks as input, calculates the average percentage, and assigns a final letter grade. This project demonstrates working with lists, loops, conditional logic, and user input validation in Python.

## ?? Features

* **Dynamic Input:** Allows the user to specify the exact number of subjects they want to calculate.
* **Input Validation:** Ensures users enter valid numbers and restricts mark entries to a realistic 0-100 range.
* **Standard Grading Scale:** * `A`: 90% - 100%
  * `B`: 80% - 89%
  * `C`: 70% - 79%
  * `D`: 60% - 69%
  * `F`: < 60%
